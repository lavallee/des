Callouts

Score bands are not a gradient.

Bucket continuous confidence values into three states. Thresholds default to ≥0.70 / ≥0.40 / <0.40 and are documented per tool.

Never use a modal for something the user wants to read alongside the list.

Use an inspector panel instead. It opens without navigating away and keeps the selected row in context.
